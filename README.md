# Handwritten Digit Recognition using CNN

This project implements a Convolutional Neural Network (CNN) using TensorFlow/Keras to recognize handwritten digits from the famous MNIST dataset. The model achieves high accuracy and includes evaluation metrics like a confusion matrix.

## 🚀 Features
* **Deep Learning Framework:** Built with TensorFlow and Keras.
* **Dataset:** MNIST dataset (60,000 training images, 10,000 testing images of 28x28 pixels).
* **Visualization:** Includes training progress graphs, evaluation metrics, and a Confusion Matrix using Seaborn.
* **Optimized Architecture:** Uses Convolutional layers, Max Pooling, Dropout (to prevent overfitting), and Dense layers.
